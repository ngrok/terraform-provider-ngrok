# ngrok Provider

The ngrok provider is used to interact with the [ngrok API](https://ngrok.com/docs/api/) to manage domains, endpoints, certificates, and other resources.

## Authentication

The provider requires an ngrok API key. You can create one in the [ngrok dashboard](https://dashboard.ngrok.com/api-keys).

Set it via the `NGROK_API_KEY` environment variable (recommended) or in the provider configuration block.

## Example Usage

```terraform
terraform {
  required_providers {
    ngrok = {
      source  = "ngrok/ngrok"
      version = "~> 1.0"
    }
  }
}

# Configure via NGROK_API_KEY environment variable (recommended)
provider "ngrok" {}

# Or configure explicitly
# provider "ngrok" {
#   api_key = "your-api-key"
# }
```
